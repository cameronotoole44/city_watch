# city_watch

### **The Problem**

> "does [character] stream?"
>
> "what is [character_name]'s stream?"

### **The Easy Solution**

find out if your favorite NoPixel character(\*) is actually **in the city.**

**city_watch** shows current live streams on **Twitch** and **Kick** so you don't have to go hunting for links.

_\*whitelisted players_

### **Features**

- see who is live on Twitch / Kick
- search for character / streamer
- uptime
- viewer count
- character name
- stream title
- list of offline streamers
- click live card to head directly to stream
